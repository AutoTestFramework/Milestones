    Q3: List some advantages and disadvantages of Automation testing:

    ATF

    R3:

      Advantages:
    • Fewer Human resources
    • Reusability
    • More Test Coverage in less time
    • Reliability
    • Parallel execution of test cases
    • Fast
      Disadvantages:
    • Development and maintenance time is more.
    • Tool Cost
    • Skilled resources are required.
    • Environment setup
    • Test Script debugging is an issue.

    Alexandru Sacerdoteanu
